这个项目是我学verilogHDL的期末大作业，具体功能如下：

题目1：基于BASYS3开发板的多功能电子密码锁设计
设计基本要求：

1.密码预设：系统内预设一个4位十进制密码（例如 1234），每位数值范围0-9，使用内部寄存器存储。

2.密码输入：用户通过拨码开关输入4位密码，每输入一位（通过按键确定），对应的LED亮起，数码管显示当前已输入的位值（如“12_34”）。

3.最终确认：输完4位后按下“确认键”，系统比对待输入密码与预设密码。
若正确：数码管显示“OPEN”（解锁），某个LED指示解锁，系统维持解锁状态5秒后自动恢复锁定。
若错误：数码管显示“ERR1”（第一次错误）、“ERR2”等，错误计数器加1。

4.清除键：按“清除键”清空当前输入的密码位，数码管归零，重新开始输入。

5. 复位键：按“复位键”强制系统回到初始状态（锁定模式，错误计数器清零，清除所有输入）。
高级要求：

6.错误次数限制：连续输错3次，系统进入锁定状态。锁定期间所有按键无效，数码管显示“LOCK”（或倒计时秒数），30秒后自动复位到待输入状态，错误计数器清零。

7.修改密码功能：必须先正确输入旧密码解锁后，再连续两次输入新密码（需一致），才能将新密码存入内部寄存器。
